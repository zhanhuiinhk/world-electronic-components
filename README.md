# World Electronic Components Database

A global, open-source repository for electronic components. Dedicated to building a comprehensive, community-driven database of component specifications, origins, and technical datasheets, enabling seamless collaborative expansion and universal accessibility for engineers and manufacturers worldwide.

## 📁 分类架构体系
本数据库参考全球主流分销商（Mouser, DigiKey, Arrow）标准，划分为以下八大类：

1. **Semiconductors** (半导体)
2. **Passive Components** (无源器件)
3. **Connectors & Switches** (连接与机电)
4. **Sensors** (传感器)
5. **RF & Wireless** (射频与无线)
6. **Optoelectronics & Displays** (光电与显示)
7. **Power & Thermal** (电源与散热)
8. **Dev Boards & Tools** (开发工具)
