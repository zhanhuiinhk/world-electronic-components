Welcome

An open database of electronic components / parts / modules from manufacturers worldwide — searchable, manufacturer-friendly, and taxonomy-extensible.

  • 🔍 Online search (GitHub Pages)
  • Taxonomy (DigiKey / Mouser / Arrow aligned)
  • Category-specific attribute profiles
  • CSV → JSON batch import
  • Manufacturer onboarding
  • Contributing overview
