Category-specific attributes

Defined in schema/attribute-profiles.json, keyed by sub_category_slug.

  • required — enforced by validate.py (e.g. resistors need resistance_ohm)
  • recommended — suggested fields
  • fields — type, unit, descriptions

Resistor example

{
  "sub_category_slug": "resistors",
  "attributes": {
    "resistance_ohm": 10000,
    "tolerance_percent": 1,
    "power_w": 0.063
  }
}
