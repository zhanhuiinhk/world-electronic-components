CSV → JSON 批量导入

工具：scripts/csv_to_json.py

用法

python scripts/csv_to_json.py examples/import-resistors-sample.csv \
  -o data/passive-components/yageo.json
python scripts/validate.py

列约定

  • 顶层字段：part_number, manufacturer, manufacturer_id, category_slug, sub_category_slug, package, datasheet_url …
  • 参数字段：attr.resistance_ohm 或 attributes.resistance_ohm → 写入 attributes
  • 标签：tags 用 | 分隔，如 resistor|smd

模板

  • examples/import-template.csv
  • examples/import-resistors-sample.csv

导入前请先在 manufacturers/ 登记厂家，并保证分类 slug 存在于 taxonomy。
