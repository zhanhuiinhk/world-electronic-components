attribute-profiles

文档首页 EN

按品类约束 attributes

定义文件：schema/attribute-profiles.json。键为 sub_category_slug。

  • required：校验必填（如电阻必须有 resistance_ohm）
  • recommended：推荐字段
  • fields：类型、单位、中英文说明

电阻示例

{
  "sub_category_slug": "resistors",
  "attributes": {
    "resistance_ohm": 10000,
    "tolerance_percent": 1,
    "power_w": 0.063
  }
}

python scripts/validate.py 会对已配置品类检查必填 attributes。
