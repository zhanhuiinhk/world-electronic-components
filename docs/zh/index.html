欢迎

本库收集全球厂家的元器件 / 部件 / 组件公开规格，支持社区检索、厂家入驻与分类扩项。

  • 🔍 在线检索（GitHub Pages）
  • 分类体系（对齐 DigiKey / Mouser / Arrow）
  • 品类专用 attributes 字段表
  • CSV → JSON 批量导入
  • 新厂家入驻通道
  • 贡献指南摘要

三层粒度

product_class含义例子
component元器件电阻、MCU、二极管
part部件线束、散热器、机箱
module组件/模组电源模块、无线模组、开发板

本地命令

python scripts/validate.py
python scripts/search.py STM32
python scripts/csv_to_json.py examples/import-resistors-sample.csv -o data/passive-components/yageo.json
python scripts/build_catalog.py
